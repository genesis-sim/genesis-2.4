PGENESIS -- Parallel GENESIS Homepage

NOTE: This older documentation from PSC may contain some invalid external links.

Parallel GENESIS was developed at the Pittsburgh Supercomputing Center to enable the well-known GENESIS neural simulator to run on a variety of parallel machines. It can be used on most Unix or Linux platforms that support PVM or MPI, from workstation clusters to supercomputers.  For users wishing to run large-scale PGENESIS simulations on supercomputers at the Neuroscience Gateway Portal (NSG), see the https://www.nsgportal.org/ web page.
